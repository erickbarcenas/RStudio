Un estadístico más/menos un margen deerror
Lo normal es que desconozcas los valores de parámetros poblacionales, de manera quetomas muestras y
utilizas estadísticos para dar la estimación posible.


Por regla general, para estimar el valor de un parámetro setoma una muestra dela población y se utilizan estadísticos de esa muestra que permitan obtener una estimación válida.

Algunas deestas estimaciones son más fáciles de obtener que otras

Supongamos que quieres saber la respuesta a una ´pregunta
hací que tomas una muestra y obtienes ciertos datos
a los que les vas sumar y restar un número para indicar queestás dando un intervalo devalores
posibles. El número que restas y sumas se llama margen de
error (ME) el cuál quiere decir que los resultados de una muestra no son exactamenteiguales a los que habrías obtenido si hubieras incluido a toda la población

El extremo inferior del intervalo es el estadístico menos el margen de
error,y el extremo superior es el estadístico más el margen deerror

.Este grado deconfianza seexpresa como porcentajey se denomina nivel de confianza.


-------------------------------------
1.Elegir el nivel de confianza y el tamaño muestral.
2. Seleccionar una muestra aleatoria de elementos de la población.
3. Recopilar datos fiables y relevantes de los elementos que componen la muestra.
4. Resumir los datos en un estadístico, por ejemplo una media o una proporción.
5. Calcular el margen de error.
6. Sumar y restar el margen de error al estadístico para obtener la estimación final del parámetro. Este paso calcula el intervalo de
confianza paraese parámetro.
